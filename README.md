# مركز 7 أكتوبر الطبي

الموقع الرسمي العربي لمركز **7 أكتوبر الطبي — October 7 Medical Center**، بهوية بصرية بترولية وذهبية، وتجربة استخدام متجاوبة تتمحور حول المريض.

## الروابط

- [فتح الموقع عبر GitHub Pages](https://alferasayman-pixel.github.io/7October-medicalcenter/)
- [تعديل التصميم المرجعي في Canva](https://www.canva.com/d/B3PhFD6RkuV04Lj)
- [معاينة التصميم المرجعي في Canva](https://www.canva.com/d/OGWp8Z9iE7uDdFN)

## التحسينات الحالية

- تصميم عربي RTL احترافي ومتجاوب مع الهاتف والكمبيوتر.
- تنقل واضح مع قائمة هاتف وشريط اختصارات سفلي.
- بحث وتصنيف تفاعلي للخدمات الطبية.
- نموذج طلب موعد مبسط مع تحقق من البيانات وتجهيز رسالة واتساب.
- أقسام لرحلة المريض، الجودة وسلامة المريض، الحقوق، والأسئلة الشائعة.
- إرشادات واضحة للحالات العاجلة وقناة للملاحظات والشكاوى.
- تحسينات الوصول: لوحة المفاتيح، رابط تجاوز المحتوى، تباين واضح، ودعم تقليل الحركة.
- استمرار إزالة Service Worker والتخزين المؤقت القديم لمنع مشكلة الصفحة الفارغة.
- تحسينات SEO وملفات Sitemap وRobots.
- النشر من فرع `gh-pages` المطابق للفرع `main`.

## بيانات يجب تحديثها قبل الإطلاق الرسمي

- رقم الهاتف ورقم واتساب داخل `data-whatsapp` في `index.html`.
- العنوان ورابط الخريطة.
- البريد الإلكتروني الرسمي.
- أسماء الأطباء وصورهم ومؤهلاتهم ومواعيدهم.
- التخصصات والخدمات النهائية المرخصة.

## ملفات المشروع

- `index.html`: الهيكل والمحتوى الدلالي.
- `styles.css`: نظام التصميم المتجاوب.
- `app.js`: التنقل، البحث، التصفية، والحجز.
- `assets/logo.svg`: شعار المركز.
